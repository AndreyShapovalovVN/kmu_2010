# UkrTranslit — KMU 2010

Транслітерація українського алфавіту латиницею відповідно до
[Постанови Кабінету Міністрів України від 27 січня 2010 р. № 55
«Про впорядкування транслітерації українського алфавіту латиницею»](https://zakon.rada.gov.ua/laws/show/55-2010-%D0%BF#Text).


## Встановлення

```bash
pip install -e .
```


## Використання

**Транслітерація одного символу** (з урахуванням контексту):

```python
from UkrTranslit import Kmu2010

t = Kmu2010()

# Літера на початку слова (previous = None)
t.translateral(None, 'Є')   # → 'YE'
t.translateral(None, 'Ї')   # → 'YI'
t.translateral(None, 'Й')   # → 'Y'
t.translateral(None, 'Ю')   # → 'YU'
t.translateral(None, 'Я')   # → 'YA'

# Літера всередині слова
t.translateral('п', 'Є')   # → 'IE'
t.translateral('а', 'Ї')   # → 'I'

# Спеціальне правило: «зг» → «zgh»
t.translateral('З', 'Г')   # → 'GH'
```

**Транслітерація повного рядка**:

```python
t.transliterate('київ')         # → 'kyiv'
t.transliterate('харків')       # → 'kharkiv'
t.transliterate('єнакієве')     # → 'yenakiieve'
t.transliterate('їжакевич')     # → 'yizhakevych'
t.transliterate('згурський')    # → 'zghurskyi'
t.transliterate('місто київ')   # → 'misto kyiv'
```

## Особливі правила

- Сполучення **«зг»** (у будь-якому регістрі) передається як **«zgh»**
  (наприклад: *Згурський* → *Zghurskyi*).
- **М'який знак (Ь)** не відображається в латинській транскрипції.
- Апостроф та інші не-українські символи передаються без змін.
- Регістр результату відповідає регістру вхідної літери.

> **Примітка.** Для слів, що починаються на Ю, Я, Є, Ї, Й, перша буква
> результату може бути двосимвольною (наприклад, **Ю → YU**). При
> посимвольному виведенні обидва символи матимуть той самий регістр, що й
> вхідна літера.
